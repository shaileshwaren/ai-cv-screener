Generative AI Full-Stack Engineer

We're building an end-to-end AI recruitment platform at Oxydata, and we need one engineer who can own the entire stack—from React dashboards to RAG pipelines to production deployment.



What You'll Build

You'll be the architect of our AI recruitment ecosystem, handling:

Frontend Development: React.js dashboards for recruiters and hiring managers with UI/UX for filters, analytics, and candidate insights
Backend Services: FastAPI web services with API endpoints, report generation, and data management
AI Processing Pipeline: Python-based cron jobs that fetch CVs, execute AI scoring via Claude/GPT, and generate intelligent candidate reports
RAG Implementation: End-to-end RAG pipelines with embeddings, vector search, semantic matching, and grounding for candidate-job fit analysis
Database Architecture: Supabase/PostgreSQL with pgvector for hybrid search (keyword + semantic)
External Integrations: ATS systems (Manatal, Zoho Recruit), Claude/OpenAI APIs, workforce platforms, and email delivery systems


Must-Haves

Education: Bachelor's in CS, Software Engineering, AI, or related field
Experience: 1–4 years building production GenAI/RAG systems with demonstrated ability to build complete applications from UI to database to AI processing
Tech Stack: 
Python (primary) + JavaScript/React
FastAPI (backend), React.js (frontend)
Supabase, PostgreSQL, pgvector (vector storage and hybrid search)
Claude API, OpenAI API, embeddings, prompt engineering
RAG Expertise: Real-world experience with chunking, embeddings, vector search (pgvector, Pinecone, Weaviate, or similar), reranking, and retrieval grounding
DevOps & Deployment: Git, REST API design, testing, error handling, cron job scheduling, and cloud platform deployment experience (Render, Vercel, Railway, AWS, GCP, or similar)
API Integration: Proven track record integrating external APIs (ATS systems, LLM providers, email services)


Nice-to-Haves

LangChain or LlamaIndex: Experience building knowledge-based RAG systems for future recruitment intelligence features (document search, historical candidate analysis, interview insights)
Experience with recruitment tech or ATS platforms (Manatal, Zoho, Greenhouse, Lever)
MCP (Model Context Protocol) or AI agent frameworks
CI/CD experience for automated testing and deployment pipelines
Docker for containerized development environments
